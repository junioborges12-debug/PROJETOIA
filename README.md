# ProjetoIA

Primeira versão navegável do aplicativo para elaboração guiada de projetos sociais e culturais.

## O que já funciona

- painel inicial responsivo;
- navegação entre Início, Projetos, Assistente IA, Editais, Aprender e Perfil;
- criação guiada de um novo projeto;
- conversa demonstrativa com preenchimento do documento ao lado;
- oportunidades com índice de compatibilidade;
- perfil institucional da Hub Cultura +;
- instalação como aplicativo web (PWA).

## Visualizar

Abra `index.html` no navegador. Para testar a instalação como aplicativo, execute um servidor local na pasta do projeto, por exemplo:

```sh
python3 -m http.server 8080
```

Depois acesse `http://localhost:8080`.
